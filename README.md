<h2>Deskripsi Proyek</h2>

<div align="justify">
<b>HospitalDB-ManagementSystem</b> merupakan proyek yang digunakan untuk implementasi perancangan basis data relasional rumah sakit sederhana menggunakan SQL. Proyek ini digunakan untuk memenuhi tugas mata kuliah <b>Manajemen Basis Data</b>.
</div>

<h2>Tujuan</h2>
<ul>
  <li>Menerapkan konsep dasar Manajemen Basis Data</li>
  <li>Memodelkan relasi antar entitas rumah sakit</li>
  <li>Menjaga integritas dan konsistensi data</li>
  <li>Menyediakan contoh query lintas tabel</li>
</ul>

<h2>Ruang Lingkup Sistem</h2>
<ul>
  <li>Data pasien, dokter, dan departemen</li>
  <li>Appointment dan rekam medis</li>
  <li>Obat dan dosis dalam rekam medis</li>
  <li>Rawat inap, ruangan, dan penugasan perawat</li>
</ul>
